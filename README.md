# alex-lang

This is a very basic language made in php.

To execute the code: launch ```php run.php```


## Example:

```
local bla = 5
local blabla = 7
print(blabla)


print(bla)
local test = 2 * 5

print(test)
```

output:
```
7
5
10
```

###Tutorial:
```local``` means that you want to add a variable.

```print``` means that you want to show a value of an variable.
